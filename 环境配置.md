## 四、Python 环境（Anaconda）

### 4.1 使用 Anaconda

项目使用 Anaconda 管理 Python 环境，环境名为 `ML`。

**创建环境**：

```bash
conda create -n ML python=3.10
conda activate ML
```

### 4.2 安装依赖

```bash
pip install thrift==0.16.0
pip install face_recognition
```

**验证**：

```bash
python -c "import face_recognition; print('OK')"
```

> 注意：thrift 版本必须为 0.16.0，新版本不支持服务端

## 六、MySQL 数据库

### 6.1 安装

**下载**：https://dev.mysql.com/downloads/mysql/

推荐 MySQL 8.0+

### 6.2 创建数据库

```sql
CREATE DATABASE face_lab DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

### 6.3 配置 Java 连接

修改 `src/java/src/main/resources/application.yml`：

```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/face_lab?useUnicode=true&characterEncoding=utf-8&serverTimezone=Asia/Shanghai
    username: root
    password: your_password
```

## 九、项目构建

### 9.1 一键构建（推荐）

```powershell
cd e:\project\face_recon_lab_person_ms
.\build.ps1
```

构建内容：

1. 下载 Qdrant 镜像
2. 安装 Python 依赖（thrift==0.16.0、face_recognition）
3. 编译 Java 项目

```powershell

conda run -n ML pip install face_recognition

# Java 构建
mvn -f src/java/pom.xml clean compile
```

## 十、项目启动顺序

### 10.1 一键启动（推荐）

```powershell
cd e:\project\face_recon_lab_person_ms
.\run.ps1
```

启动内容：

1. 启动 Qdrant 容器（端口 6333、6334）
2. 启动 Python Thrift 服务（9090）
3. 启动 Java 后端（8080）
4. 启动 Vue 前端（5173）

### 10.2 手动分步启动

**步骤1 - 启动 Qdrant（最先启动）**：

```bash
docker run -d --name qdrant -p 6333:6333 -p 6334:6334 qdrant/qdrant
```

**步骤2 - 启动 Python 服务**：

```bash
cd e:\project\face_recon_lab_person_ms
conda run -n ML python src/py/main.py
```

服务地址：`localhost:9090`

**步骤3 - 启动 Java 后端**：

```bash
cd e:\project\face_recon_lab_person_ms\src\java
mvn spring-boot:run
```

服务地址：`localhost:8080`

**步骤4 - 启动 Vue 前端**：

```bash
cd src/frontend
npm install
npm run dev
```

服务地址：`localhost:5173`

### 10.3 服务地址

## 十二、常见问题

### 12.1 dlib 安装失败

需要先安装 CMake：

```bash
pip install cmake
```

然后重新安装 dlib：

```bash
pip install dlib
```

### 12.2 Maven 依赖下载慢

可以配置阿里云镜像，在 `Maven_HOME\conf\settings.xml` 中添加：

```xml
<mirrors>
  <mirror>
    <id>aliyun</id>
    <mirrorOf>*</mirrorOf>
    <url>https://maven.aliyun.com/repository/public</url>
  </mirror>
</mirrors>
```

### 12.3 Thrift 生成代码路径问题

生成代码默认在当前目录的 `gen-py` 和 `gen-java`，需要手动移动到对应目录。

### 12.4 PowerShell 中文乱码

在 PowerShell 中执行：

```powershell
chcp 65001
```

### 12.5 Conda 激活报错 UnicodeEncodeError

使用 `conda run` 代替 `conda activate`：

```powershell
conda run -n ML python src/py/main.py
```

### 12.6 Qdrant 连接失败

检查 Qdrant 容器是否启动：

```bash
docker ps | grep qdrant
```

如果未启动：

```bash
docker start qdrant
```
