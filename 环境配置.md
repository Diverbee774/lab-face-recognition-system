## 三、Maven 环境

### 3.1 安装

**Maven 下载**：https://maven.apache.org/download.cgi

解压到目录，如 `D:\Maven\apache-maven-3.9.5`

### 3.2 配置环境变量

1. 新建系统变量：
   - 变量名：`MAVEN_HOME`
   - 变量值：`D:\Maven\apache-maven-3.9.5`

2. 编辑系统变量 `Path`，添加：
   ```
   %MAVEN_HOME%\bin
   ```

### 3.3 验证
```bash
mvn -v
```

**当前版本**：
```
Apache Maven 3.9.5
Maven home: D:\Maven\apache-maven-3.9.5
```

## 四、Python 环境

### 4.1 使用 Anaconda

项目使用 Anaconda 管理 Python 环境，环境名为 `ML`。

**创建环境**：
```bash
conda create -n ML python=3.10
conda activate ML
```

### 4.2 安装依赖

```bash
pip install face_recognition
pip install dlib
pip install thrift
```

**验证**：
```bash
python -c "import face_recognition; print('OK')"
```

## 五、Thrift 编译器

### 5.1 安装

1. 下载：https://dlcdn.apache.org/thrift/0.22.0/thrift-0.22.0.exe
2. 放到项目根目录：`e:\project\face_recon_lab_person_ms\thrift-0.22.0.exe`

### 5.2 生成代码

**Python**：
```bash
cd e:\project\face_recon_lab_person_ms
thrift-0.22.0.exe --gen py src/idl/face.thrift
```

**Java**：
```bash
cd e:\project\face_recon_lab_person_ms
thrift-0.22.0.exe --gen java src/idl/face.thrift
```

## 六、MySQL 数据库

### 6.1 安装

**下载**：https://dev.mysql.com/downloads/mysql/

推荐 MySQL 8.0+

### 6.2 创建数据库

```sql
CREATE DATABASE face_lab DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

### 6.3 配置 Java 连接

修改 `src/java/src/main/resources/application.yml`：
```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/face_lab?useUnicode=true&characterEncoding=utf-8&serverTimezone=Asia/Shanghai
    username: root
    password: your_password
```

## 九、常见问题

### 9.1 dlib 安装失败

需要先安装 CMake：
```bash
pip install cmake
```

然后重新安装 dlib：
```bash
pip install dlib
```

### 9.2 Maven 依赖下载慢

可以配置阿里云镜像，在 `Maven_HOME\conf\settings.xml` 中添加：
```xml
<mirrors>
  <mirror>
    <id>aliyun</id>
    <mirrorOf>*</mirrorOf>
    <url>https://maven.aliyun.com/repository/public</url>
  </mirror>
</mirrors>
```

### 9.3 Thrift 生成代码路径问题

生成代码默认在当前目录的 `gen-py` 和 `gen-java`，需要手动移动到对应目录。
