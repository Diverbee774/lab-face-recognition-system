## 1\. 接入说明 (General)

### 1.1 通信协议

  * **传输协议**：HTTP/1.1 (RESTful)
  * **编码格式**：UTF-8
  * **基础路径 (Base URL)**：
      * 生产环境：`https://api.lab-system.com/v1`
      * 测试环境：`http://127.0.0.1:8080/v1`

### 1.2 鉴权机制 (Authentication)

除 `Auth` (登录/注册) 模块下的接口外，所有请求均需在 HTTP Header 中携带 JWT Token。

| Key | Value | 说明 |
| :--- | :--- | :--- |
| `Authorization` | `Bearer <Your_Token_String>` | Bearer 认证方案 |

### 1.3 响应规范 (Response Schema)

所有接口统一返回如下 JSON 结构：

```json
{
  "code": 200,          // 业务状态码：200成功，4xx客户端错误，5xx服务端错误
  "msg": "操作成功",    // 状态描述
  "data": {}            // 业务数据载体
}
```

## 2\. 管理后台接口 (Admin APIs)

**权限说明**：本节接口仅限角色为 `ADMIN` 的用户调用。

### 2.1 用户管理 (User Management)

#### 2.1.1 获取用户列表

  * **URL**: `/admin/users`
  * **Method**: `GET`
  * **描述**: 分页查询系统内所有注册用户。

**Query Parameters:**
| 参数名 | 类型 | 必填 | 说明 |
| :--- | :--- | :--- | :--- |
| `page` | Int | 否 | 页码 (默认1) |
| `size` | Int | 否 | 每页条数 (默认10) |
| `keyword` | String | 否 | 搜索关键词 (学号/姓名) |

**Response Data:**

```json
{
  "total": 100,
  "list": [
    {
      "uid": "2021001",
      "name": "张三",
      "status": 1, // 1:正常, 0:禁用
      "has_face": true, // 是否已录入底库
      "reg_time": "2025-01-01"
    }
  ]
}
```

#### 2.1.2 修改用户状态

  * **URL**: `/admin/users/{uid}/status`
  * **Method**: `PUT`
  * **描述**: 禁用或启用用户。禁用后无法刷脸通行。

**Request Body:**

```json
{
  "status": 0  // 1:启用, 0:禁用
}
```

#### 2.1.3 删除用户

  * **URL**: `/admin/users/{uid}`
  * **Method**: `DELETE`
  * **描述**: 彻底删除用户数据及人脸底库文件，并触发 Python 特征库重载。

#### 2.1.4 重置用户密码

  * **URL**: `/admin/users/{uid}/password`
  * **Method**: `PUT`
  * **描述**: 将用户密码重置为默认值 (如 `123456`)。

### 2.2 实验室资源管理 (Lab Management)

#### 2.2.1 新增/编辑实验室

  * **URL**: `/admin/labs` (新增) / `/admin/labs/{id}` (编辑)
  * **Method**: `POST` / `PUT`

**Request Body:**

```json
{
  "name": "305大数据实验室",
  "location": "科技楼3F",
  "capacity": 40,
  "status": 1,      // 1:开放, 0:维护
  "mode": "PRIVATE" // 访问策略: PUBLIC (公共), PRIVATE (私有/白名单)
}
```

#### 2.2.2 实验室列表查询

  * **URL**: `/admin/labs`
  * **Method**: `GET`
  * **Response Data:**

<!-- end list -->

```json
[
  {
    "id": 1,
    "name": "公共机房",
    "mode": "PUBLIC", // 标记为公共
    "user_count": 0   // 公共实验室此字段无意义
  },
  {
    "id": 2,
    "name": "导师课题组",
    "mode": "PRIVATE",
    "user_count": 15  // 当前白名单内人数
  }
]
```

### 2.3 白名单权限管理 (Whitelist Management)

**适用场景**：仅针对 `mode = PRIVATE` 的实验室。

#### 2.3.1 获取某实验室白名单

  * **URL**: `/admin/labs/{lab_id}/whitelist`
  * **Method**: `GET`
  * **描述**: 获取有权限进入该实验室的所有用户列表。

**Response Data:**

```json
[
  { "uid": "2021001", "name": "李四", "add_time": "2025-05-20" }
]
```

#### 2.3.2 添加人员到白名单 (授权)

  * **URL**: `/admin/labs/{lab_id}/whitelist`
  * **Method**: `POST`
  * **描述**: 批量授权。

**Request Body:**

```json
{
  "uids": ["2021001", "2021002", "2021005"] // 用户ID数组
}
```

#### 2.3.3 移除人员权限 (撤权)

  * **URL**: `/admin/labs/{lab_id}/whitelist/{uid}`
  * **Method**: `DELETE`

## 3\. 用户端/终端接口 (User & Terminal APIs)

### 3.1 基础服务

  * **注册**: `POST /auth/register` (参数: 学号, 姓名, 密码)
  * **登录**: `POST /auth/login` (返回 Token)
  * **人脸底库上传**: `POST /user/face` (参数: file)

### 3.2 智能通行 (Smart Access - 终端专用)

#### 3.2.1 刷脸通行校验

  * **URL**: `/access/check`
  * **Method**: `POST`
  * **Content-Type**: `multipart/form-data`
  * **描述**: 门禁终端调用。后端负责串联 Python 识别与数据库权限校验。

**Request Parameters:**
| 参数名 | 类型 | 必填 | 说明 |
| :--- | :--- | :--- | :--- |
| `image` | File | 是 | 摄像头抓拍图片 |
| `lab_id` | Int | 是 | 当前所在实验室ID |

**Response Body (JSON):**

**场景 1：允许通行**

```json
{
  "code": 200,
  "msg": "识别成功",
  "data": {
    "allow": true,
    "uid": "2021001",
    "name": "张三",
    "tips": "欢迎进入"
  }
}
```

**场景 2：拒绝 (陌生人/识别失败)**

```json
{
  "code": 200,
  "msg": "识别失败",
  "data": {
    "allow": false,
    "reason_code": "UNKNOWN_USER",
    "tips": "未识别身份，请重试"
  }
}
```

**场景 3：拒绝 (无权限 - 不在白名单)**

```json
{
  "code": 200,
  "msg": "权限不足",
  "data": {
    "allow": false,
    "uid": "2021001",
    "name": "张三",
    "reason_code": "NOT_IN_WHITELIST",
    "tips": "您无权进入该私有实验室"
  }
}
```

**场景 4：拒绝 (账号被禁用)**

```json
{
  "code": 200,
  "msg": "账号异常",
  "data": {
    "allow": false,
    "reason_code": "USER_BANNED",
    "tips": "您的账号已被禁用，请联系管理员"
  }
}
```

## 4\. 内部协同接口 (Internal: Java -\> Python)

**服务说明**: 该模块仅限 Java 后端调用 Python AI 服务。
**Host**: `http://localhost:8000`

### 4.1 图像识别 (Recognize)

  * **URL**: `/recognize`
  * **Method**: `POST`
  * **Request Body**:
    ```json
    { "path": "/data/snapshot/20251205_uuid.jpg" }
    ```
  * **Response Body**:
    ```json
    {
      "code": 200,
      "data": {
        "uid": "2021001", // 识别到的用户ID，未识别则为 null
        "score": 0.38     // 欧氏距离
      }
    }
    ```

### 4.2 特征库重载 (Reload)

  * **URL**: `/reload`
  * **Method**: `POST`
  * **Request**: `{}`
  * **Response**: `{"code": 200, "msg": "Reloaded 120 faces"}`
